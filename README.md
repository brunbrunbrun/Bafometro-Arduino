# Bafometro_Arduino
Projeto de um bafometro feito em arduino para a materia de Arquitetura de Computadores I 2021/2

## Partes Utilizadas
 - 1 Arduino Uno R3
 - 1 LCD 16x2
 - 1 Piezo Buzzer
 - 1 LED vermelho
 - 1 LED verde
 - 1 Potenciometro de 10KΩ
 - 1 Sensor de Gás MQ-3 Álcool
 - 3 Resistores de 300Ω
